About

BiTGApps is a customized Google Apps Package to sideload Google Apps in Custom Android OS. It comes with many useful features and easy installation. Configuration file can be used to customize BiTGApps installation and more below.

Take note that BiTGApps does not provide you with any license for Google's APKs included in the package. The BiTGApps packages merely provide a convenient way to sideload APKs to your device. It is your own responsibility to obtain the proper permissions by e.g. buying an OHA-licensed device with pre-installed Google Apps and/or acquiring the applications from Google's Play Store.

The pre-built packages from BiTGApps.io are provided ONLY as courtesy by BiTGApps.io and is without warranty of ANY kind, under the terms that they can be freely used for personal use only and is not allowed to be mirrored to the public other than author. You are not allowed to modify pre-built packages for further (re)distribution.

Features

  • All Platforms and All Android Versions are Supported
  • BiTGApps provide flashable packages in two variations
    • Standard BiTGApps Package is minimal which includes the Core Google Apps like Google Play Services (GMS), Google Services Framework (GSF), Google Play Store and other essentials. Variants like Basic and Omni which includes everything from Standard with more useful Google Apps like Dialer, Contacts, Messages, etc. Variants have different set of Google Apps. Checkout list of Google Apps in Standard and Variants below.
  • BiTGApps comes with freeware Addons for Standard Package
  • BiTGApps offers Seamless OTA Updates through Survival Script
    • OTA survival script allows you to update your current ROM to its latest version without worrying about having to flash Google Apps again. This functionality gives you flash and forget experience. Currently BiTGApps using ADDOND Version 3 which is latest.
  • BiTGApps comes with Dialer and Maps Framework Support
  • BiTGApps replace AOSP Apps with Google Apps

Installation

Installing for the first time ?

Google Apps should be installed via recovery immediately after installing Custom ROM. If you reboot into system before installing Google Apps, you must factory reset and then install them, otherwise expect crashes. Addons, if you want to use, should be installed in same manner on top of BiTGApps. SetupWizard need to be configured before installation. Checkout instructions for SetupWizard below.

  • Additional Packages

Dirty Flash

Installing for the second time and so on ?

Re-flashing of same package or upgrading to latest release doesn't require factory reset.


SetupWizard

  • SetupWizard's main purpose is to assist Android users when they have to first set up their phones, especially if they want to transfer old data, such as apps, pictures, and personalized features, from their previous phone or right after a reset.

Want to install SetupWizard with BiTGApps ?

In order to have SetupWizard with BiTGApps, config file must be placed in internal storage outside of folder before installation. Avoid placing config file in folder that contain special characters like "@#$" and spaces in their name. LineageOS Recovery users needs to follow below workaround.

  • Download Config

Workaround

With LineageOS Recovery and similar, Adding configuation file to internal storage will not work. Because of encrypted data partition. You need to re-pack BiTGApps ZIP with config file. The file must goes in root of un-packed BiTGApps ZIP else it won't work.


Manual Creation ?

This is BiTGApps configuration file.

  • bitgapps-config.prop

Below property goes in configuration file.

  • ro.config.setupwizard=true
