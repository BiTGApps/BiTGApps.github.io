Features

  • All Platforms and All Android Versions are Supported
  • BiTGApps provide flashable packages in two variations
    • Standard BiTGApps Package is minimal which includes the Core Google Apps like Google Play Services (GMS), Google Services Framework (GSF), Google Play Store and other essentials. Variants like Basic and Omni which includes everything from Standard with more useful Google Apps like Dialer, Contacts, Messages, etc. Variants have different set of Google Apps. Checkout list of Google Apps in Standard and Variants.
  • BiTGApps comes with freeware Addons for Standard Package
  • BiTGApps offers Seamless OTA Updates through Survival Script
    • OTA survival script allows you to update your current ROM to its latest version without worrying about having to flash Google Apps again. This functionality gives you flash and forget experience. Currently BiTGApps using ADDOND Version 3 which is latest.
  • BiTGApps comes with Dialer and Maps Framework Support

Installation

Installing for the first time ?

Google Apps should be installed via recovery immediately after installing Custom ROM. If you reboot into system before installing Google Apps, you must factory reset and then install them, otherwise expect crashes. Addons, if you want to use, should be installed in same manner on top of BiTGApps. SetupWizard need to be configured before installation.

  • List of GApps Packages
  • Additional Packages
  • Configure SetupWizard

Dirty Flash

Installing for the second time and so on ?

Re-flashing of same package or upgrading to latest release doesn't require factory reset.


SetupWizard

Want to install SetupWizard with BiTGApps ?

In order to have SetupWizard with BiTGApps, config file must be placed in internal storage outside of folder before installation. Avoid placing config file in folder that contain special characters like "@#$" and spaces in their name.

  • Download Config

Workaround for LineageOS

With LineageOS Recovery, Adding configuation file to internal storage will not work. Because of encrypted data partition. You need to re-pack BiTGApps ZIP with config file. The file must goes in root of un-packed BiTGApps ZIP else it won't work.
